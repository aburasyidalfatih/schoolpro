## 🎯 Tentang SISPRO

SISPRO adalah platform Sistem Informasi Sekolah berbasis web yang dirancang untuk mengelola seluruh operasional sekolah secara digital — mulai dari data master, keuangan (tagihan, pembayaran, arus kas), tabungan siswa, PPDB Online, hingga portal berita & pengumuman.

### 3 Portal Utama
- **🔧 Dasbor Petugas** — Admin, Keuangan, Tata Usaha, Staf
- **👨‍👩‍👧‍👦 Dasbor Wali/Siswa** — Cek tagihan, tabungan, informasi
- **📝 Portal PPDB Online** — Pendaftaran siswa baru

### Keunggulan Arsitektur
- **Multi-Tenant** — Satu instalasi untuk banyak sekolah (SaaS-Ready)
- **DB-Agnostic** — MySQL sekarang, PostgreSQL nanti (via Prisma ORM)
- **Modern Stack** — Next.js 15, TypeScript, Server Actions
